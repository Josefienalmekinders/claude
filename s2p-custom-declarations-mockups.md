## 12. Comparison: PAX Module vs. Customs Module

| Aspect | PAX Declaraties | Customs Declaraties |
|--------|----------------|---------------------|
| Overview cards | Totaal / Goedgekeurd / In behandeling / Totaal PAX | Totaal / Goedgekeurd / In behandeling / Totaal items |
| Configuration button | EORI-nummers configureren | Customs configuratie |
| Tab structure | Single view with person list | 5 tabs (General, Parties, Master Consignment, Items, IRP Envelopes) |
| Line items | Persons (crew/passengers) | Cargo items (goods/commodities) |
| Item fields | Name, nationality, gender, DOB, ID type/nr | Description, HS code, weight, packages, TE type |
| Actions | Persoon toevoegen, Upload lijst, ISPS-verzoek | Item toevoegen, Exporteer bestand, Verstuur declaratie |
